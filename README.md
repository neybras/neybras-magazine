# Neybras Magazine

Revue juridique & financière — site statique HTML/CSS/JS.

## Structure

```
neybras-magazine/
├── index.html                      # Page d'accueil
├── droit.html                      # Rubrique Droit des affaires
├── finance.html                    # Rubrique Finance & Marchés
├── arbitrage.html                  # Rubrique Arbitrage & ADR
├── afrique.html                    # Rubrique Afrique & OHADA
├── droit-du-sport.html             # Rubrique Sport Business
├── interviews.html                 # Interviews
├── tribunes.html                   # Tribunes libres
├── contact.html                    # Contact & Soumissions
├── article.html                    # Template article générique
├── article-zlecaf.html             # ZLECAf & harmonisation juridique
├── article-finance-participative.html  # Finance participative loi 103-12
├── article-procedures-collectives.html # Procédures collectives loi 73-17
├── article-cirdi-maroc.html        # CIRDI & TBI marocains
├── article-ia-cndp.html            # IA & CNDP loi 09-08
├── article-code-travail.html       # Réforme Code du travail
├── article-cfc-nairobi.html        # CFC vs Nairobi (hubs africains)
├── article-green-bonds.html        # Green Bonds & Sukuk verts
└── images/                         # Images locales
```

## Technologies

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts : Cormorant Garamond, Playfair Display, Inter
- Font Awesome 6.5 (CDN)
- Images : Unsplash CDN

## Déploiement GitHub Pages

1. Pousser sur la branche `main`
2. Aller dans **Settings → Pages**
3. Source : `Deploy from a branch` → `main` → `/ (root)`
4. Le site sera disponible à `https://<username>.github.io/<repo>/`
